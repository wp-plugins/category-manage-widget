This plugin provides full featured category management widget with multiple instance and configurable options.

== Description ==
This is a full featured multiple instance supported category manage widget with configurable options to control parameters like category hierarchy, order category,include/exclude category etc features in your sidebar.This plugin displays the categories using wordpress wp_list_categories function.

**Features**
 
* Multiple instance of the widget can be created
* Easy to use widget control panel
* Multiple category parameter configuring options
* and many more

== Installation ==

1. Download "category-manage-widget" plugin
2. Open /wp-content/plugins folder at your web server
3. Upload the folder "category-manage-widget" there
4. Goto Plugins page in your wordpress back office
5. Activate "Category Manage Widget" plugin
6. Go to Settings >> Category Manage Widget
6. Go to Appearance >> Widgets
7. Drag your plugin to the right sidebar to activate your plugin
8. Note: This plugin will require one time free registration

== How to use it ==

* Go to Widgets in your admin panel > drag the Category Manage Widget in the right sidebar
* Click in right side of the widget to open the configuration panel
* In the title section enter the title you want to be appeared in the front end sidebar at the top of your category listing
* Child of option can be used if you want to list child category of a parent category,parent category can be selected from the select box
* Feed option can be used to enter text to appear as a rss feed link next to the category in frontend
* Feed Image options is same as Feed but in this case a image can be placed to link the rss feed
* From Exclude Category option you can select multiple category you can exclude from your category listing
* Note:to select multiple categories use control key
* Include category is same as Exclude but here you select it to include the category in the listing
* Title li option is used to display a li text just before the category hierchial order list style
* From Sort Options you can select the style type to be used in the category listing
* Hierarchical options lists category in a parent child manner
* Show Post Counts displays the total post in a category in a bracket
* Use Desc for Title is used to set hyperlink title with category description
* Hide Empty Categories options is uded to hide category with no posts
